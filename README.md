# Laboratoire de Sécurité des Applications
Ce repository contient les laboratoires pratiques du module **Sécurité des Applications**.
Les travaux pratiques couvrent notamment :
- SOP (Same-Origin Policy)
- CORS
- Upload de fichiers
- XSS
- JWT
- Cookies
- Architecture Web
- Contrôle d'accès

Les laboratoires seront publiés progressivement selon l'avancement du cours.
